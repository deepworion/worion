# ANALIZES

Arquivos movidos para revisao antes de decidir se devem ser versionados, arquivados fora do repositorio ou descartados.

## Categorias

- `root-docs/`: changelogs, instrucoes e READMEs gerados durante frentes recentes.
- `artifacts-docs/`: documentacao operacional gerada em `artifacts/`.
- `artifacts-audits/`: auditorias, sweeps e evidencias de diagnostico.
- `artifacts-mds/`: markdowns/textos gerados em `artifacts/mds`, exceto `WORION_VOICE.md`, que continua no runtime.
- `artifacts-images/`: imagens geradas em `artifacts/images`.
- `diagnostics/`: scripts e resultados de diagnostico pontual.

## Regra

Nada aqui foi apagado. Esta pasta e uma quarentena para revisao manual antes de qualquer decisao final.
