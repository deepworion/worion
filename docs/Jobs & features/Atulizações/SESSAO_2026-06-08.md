#### **FASE 6: Correção de Erro (Agente 6 - Correção)**

**Problema**: `ReferenceError: buildSystemPrompt is not defined` em chat.js:2295

**Diagnóstico**:
- `buildSystemPrompt()` não tinha verificação de tipo antes de ser chamada
- Erro secundário: backticks não escapados em template strings em prompt.js

**Correção aplicada**:
```javascript
// ANTES:
const systemPrompt = await buildSystemPrompt(...);

// DEPOIS:
const systemPrompt = typeof buildSystemPrompt === 'function'
  ? await buildSystemPrompt(...)
  : `Sistema indisponivel...`;
```

**Validações**:
- `node --check js/chat.js` ✅
- `node --check js/prompt.js` ✅
- `npm start` iniciado sem erros ✅
- Logs do Electron limpos ✅
